# Multimodal Writing Portfolio Template

This repository includes a single-page **multimodal writing portfolio website** styled like a **Canva / Google Slides** presentation.

## Files

- `index.html` — portfolio structure and content sections
- `style.css` — slide-style layout, typography, colors, cards, responsiveness
- `script.js` — navigation dots, smooth section scrolling, tabs, reveal animations

## How to Customize

1. Open `index.html`.
2. Replace all placeholders:
   - `[Your Name]`
   - `[Course Name & Section]`
   - `[Semester / Year]`
   - Draft and reflection placeholders
   - Media captions and source credits
3. For essay drafts, paste your writing into each draft panel:
   - Initial Draft
   - Submitted Draft
   - Revised Draft
   - Revision Explanation paragraph
4. Replace gallery images if desired (currently using `picsum.photos` placeholders).
5. Replace the sample media embed in the Media Gallery section:
   - Swap the YouTube `iframe` with your own video/audio link, or use an HTML5 `<video>`/`<audio>` tag.
6. Replace the infographic placeholder card with your own graphic organizer image or embed.

## GitHub Pages Deployment (Live Portfolio Link)

To publish this site:

1. Push changes to the `main` branch.
2. In GitHub, open **Settings → Pages**.
3. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
4. Save settings and wait for deployment.

Your live link format will be:

`https://anthonyalejos10324-prog.github.io/Ant/`

## Rubric Alignment

- **Design & Clarity (30 pts):**
  - Slide-based layout, visual hierarchy, readable cards, responsive design
- **Revisions w/Explanation (90 pts):**
  - Essay 1, Essay 2, Essay 3 each include:
    - Initial Draft
    - Submitted Draft
    - Revised Draft
    - Revision Explanation paragraph
- **Final Reflection Essay (75 pts):**
  - Dedicated reflection section with all 3 required reflection topics
