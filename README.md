# Multimodal Portfolio Editor

A Canva/Google Slides-style single-page portfolio editor built with plain HTML, CSS, and vanilla JavaScript.

## Open the editor
- Local: open `index.html` in any desktop browser.
- GitHub Pages: publish this repository and visit the Pages URL.

## Use the editor
- **Slides:** Use the slide panel on the left to switch between the **Cover**, **Essay 1**, **Essay 2**, and **Essay 3** slides.
- **Edit text:** Click text blocks directly and type inline.
- **Add text:** Click **Add Text** to insert a new draggable text box.
- **Add image:** Click **Add Image** and choose a local file to insert a draggable/resizable image.
- **Move elements:** Select an element, then drag using the small move handle above it.
- **Resize elements:** Select an element and drag corner handles.
- **Delete elements:** Select an element and press **Delete** (when not typing), or click **Delete** in the toolbar.
- **Text formatting:** Use **Bold**, **Italic**, **Underline**, **Size**, and **Text Color** in the toolbar.
- **Slide background:** Use the **Background** color picker for the current slide.

## Save and export
- **Auto-save:** Changes are automatically saved to `localStorage`.
- **Manual save:** Click **Save** to explicitly save the current state.
- **Export:** Click **Export** to download a self-contained HTML file of your portfolio.
